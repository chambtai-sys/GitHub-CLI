# GitHub-CLI

This is a terminal version of GitHub, designed to bring the power of GitHub to your command line.

## Features

*   **Repository Management:**
    *   `repo list`: List your repositories.

## Usage

To use the GitHub CLI, run the `gh.py` script with the desired command:

```bash
python gh.py repo list
```
